Background position

Sets the background position of images. A static declaration of "no-repeat" is implicitly included.

Static declarations

  • background-repeat: no-repeat;

A static declaration is not altered by the provided values.

Dynamic properties

  • .bgp
    • background-position

The value of a dynamic property is determined by the provided values.

Provided values

left top
