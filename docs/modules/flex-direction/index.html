Flex direction

Establishes the main-axis, thus defining the direction flex items are placed inside the flex container.

Examples

No examples yet.

Dynamic properties

Dynamic properties combine with every single one of the module values to make declarations. They form declaration blocks together with the optional static declarations.

base selector properties
.fd flex-direction

Module values

The module values consist each of a suffix and a value. The suffix is attached to the base selector and the value forms a declaration with every dynamic property.

suffix value
row row
col row-reverse
rowr column
colr column-reverse

Media

prefix 0 36 48 62 75
(none)

CSS rules


.fd-row { flex-direction: row; }
.fd-col { flex-direction: row-reverse; }
.fd-rowr { flex-direction: column; }
.fd-colr { flex-direction: column-reverse; }
